# GitHub- und Netlify-Start

## 1. Dateien in GitHub hochladen

1. Das Repository `onlang-tv` öffnen.
2. **Add file → Upload files** wählen.
3. Den Inhalt dieses entpackten Ordners vollständig in das Upload-Feld ziehen.
4. Commit-Nachricht: `ONLANG TV Präsentationsversion 1.0`
5. **Commit changes** anklicken.

Wichtig: `index.html` muss anschließend direkt auf der obersten Ebene des Repositorys liegen, nicht in einem zusätzlichen Unterordner.

## 2. Mit Netlify verbinden

1. Netlify öffnen und **Add new project → Import an existing project** wählen.
2. GitHub auswählen und das Repository `onlang-tv` verbinden.
3. Build command leer lassen.
4. Publish directory: `.`
5. Veröffentlichung starten.
